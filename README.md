# SGPA & CGPA Calculator

A modern, responsive, and user-friendly **SGPA & CGPA Calculator** built using **HTML, CSS, and JavaScript**. This tool allows students to add multiple semesters, enter subject details, select grades, assign credits, and instantly calculate semester-wise SGPA and overall CGPA. 

## 🚀 Features

* Add unlimited semesters
* Add multiple subjects for each semester
* Enter subject credits
* Select grades using predefined grade points
* Calculate:

  * Semester-wise SGPA
  * Overall CGPA
  * Total Semesters
  * Total Subjects
  * Total Credits
* Performance badge based on CGPA
* Reset all data instantly
* Fully responsive design for desktop and mobile
* Clean and modern UI

## 📖 How to Use

1. Open the website.
2. Add semesters if required.
3. Add subjects under each semester.
4. Enter:

   * Subject Name
   * Credits
   * Grade
5. Click **Calculate CGPA**.
6. View:

   * Semester SGPA
   * Total Credits
   * Total Subjects
   * Overall CGPA
   * Performance Status
7. Use **Reset All** to start again.
